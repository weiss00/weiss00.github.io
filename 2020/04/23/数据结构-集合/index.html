数据结构-集合

发表于 2020-04-23 分类于 编程基石

集合(Set)

特点

  • 集合中没有重复的元素 – 可以用来去重

底层实现

  • 利用链表（LinkedList）来实现 (无序的集合)
    • 注： 利用链表实现集合时， 对于插入元素需要判断是否重复添加
  • 利用二分搜索树（BST）实现 (有序的集合)

时间复杂度计算

  • BST实现的集合

    ( h 为BST的深度 )BSTSet平均最坏
    增 addO(h)O(logn)O(n)
    查 containsO(h)O(logn)O(n)
    删 removeO(h)O(logn)O(n)

  • 平均时间复杂度

  • 综合上图如果BST为满二叉树， 则h层节点的个数为（2^h-1）个
  • 当节点个数为n ， 2^h-1 = n ==> h = O(logn)
  • 最坏时间复杂度
    • 如图所示， 几乎等同于链表

  • LinkedList实现的集合

    • 增 O(n)
    • 删 O(n)
    • 查 O(n)

  • 相关练习

    • LeetCode 804题目
      • 思路： 利用集合做到去重的效果， 并且巧妙利用（字母 - ‘a’）获取对应的摩斯密码字符串索引。
