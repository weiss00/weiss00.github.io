JavaScript正则表达式

注： 所有的表达式 ^前后都没有空格 表达式 别名
匹配字母a-zA-Z和数字0-9以及下划线 /[a-zA-Z0-9_]/ /\w/
^匹配字母a-zA-Z和数字0-9以及下划线 /[ ^a-zA-Z0-9_ ]/ /\W/
除了换行符（\n）以外全部都能匹配 /[ ^\n ]/ /./
匹配0~9数字 /[0-9]/ /\d/
^匹配0~9数字 /[ ^0-9 ]/ /\D/
匹配制表符，换行符，unicode空白字符 /\s/
^匹配制表符，换行符，unicode空白字符 /\S/
匹配元素的个数num / 个数 num >num1, num < num2 {num} / {num1, num2}
匹配元素的个数至少为num个 {num, }
匹配元素的个数为0个或者1个 {0， 1} ？
匹配元素的个数至少为1个 {1，} +
匹配元素的个数为0个或者更多个 {0，} *
忽略大小写匹配 i
全局匹配 g
多行匹配 m
