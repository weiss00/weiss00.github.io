二叉树

  • 图示

二叉树具有天然递归结构

  • 每个节点的左子树也是一棵二叉树
  • 每个节点的右子树也是一棵二叉树

一个结点也是二叉树

NULL 空也是二叉树

二分搜索树 Binary Search Tree(BST)

  • 二分搜索树也是二叉树
  • 二分搜索树的每个节点的值:
    • 大于其左子树的所有结点的值
    • 小于其右子树的所有结点的值
  • 每一棵子树也是二分搜索树
  • 注意： 存储的元素必须有可比较性

  • 向二分搜索树插入元素

  • 二分搜索树查询元素存在与否

二分搜索树的遍历

  • 二分搜索树（前序遍历）
    • 先查看节点， 之后对左右子树进行遍历
    • 由于遍历时会先到达树的最深处， 所以又叫 “深度优先遍历 “

  • 二分搜索树 （中序遍历）

    • 先对左子树进行遍历， 然后查看节点， 最后遍历右子树
    • 二分搜索树的中序遍历的结果是有顺序的

  • 二分搜索树 （后序遍历）

    • 先对左右子树进行遍历，最后查看节点

  • 二分搜索树的层序遍历 （广度优先遍历）
