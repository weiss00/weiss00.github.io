发表于 2020-05-13 分类于 前端学习

ES6

JavaScript与ES6关系

  • ES6是一种规范标准
  • JavaScript则是该标准的实现

Let

  • let只作用于当前块级作用域内
  • 使用let或者const声明的变量 不能在被重新声明
  • question: 生成十个按钮， 每个按钮点击的时候弹出 1 ~ 10

const

  • 修饰常量的关键字
  • 常量不可被修改 否则报 TypeError: Assignment to constant variable. 错误
  • 常量必须在声明的时候赋值 否则报 Missing initializer in const declaration 错误
  • 常量为引用类型(Object， Array….)的时候， 可以修改该引用类型

  • 那么问题来了 怎样防止常量为引用类型的时候能被修改的情况

    • 使用 Object.freeze() 方法后 该引用类型 无法被修改， 也无法被扩展

模板字符串

  • 反引号 ``

字符串部分新方法

  • padStart、 padEnd

    • padStart 向前补位

    • padEnd 向后补位

  • repeat 复制

    • 不要传入负数， 传入小数一律取整

  • startsWith、 endsWith

    • startsWith 查询以XXX开头
    • endsWith 查询以XXX结尾

  • includes

    • 判断是否存在某个字符串
