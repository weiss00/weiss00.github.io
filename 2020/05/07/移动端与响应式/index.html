视窗声明

媒体查询 （@media）

  • @media only screen and …

    • only(限定某种设备)

    • screen 是媒体类型的一种

    • 常用类型

      all 所有设备
      braille 盲文
      embossed 盲文打印
      handheld 手持设备
      print 文档打印或打印预览模式
      projection 项目演示，比如幻灯
      screen 彩色电脑屏幕
      speech 演讲
      tty 固定字母间距的网格的媒体，比如电传打字机
      tv 电视

    • and 被称为关键字，其他关键字还包括 not(排除某种设备)

  • screen用的比较多

    • 常用设备

      设备 屏幕尺寸
      显示器 1280 x 800
      ipad 1024 x 768
      Android 800 x 480
      iPhone 640 x 960
  • 整活运动
